# MingChengAPI开发文档

MingChengAPI是一个高自由度，开放的Vela第三方快应用API（不止Vela）

## 快速接入

MingChengAPI提供快速，简洁的接入方法

### STEP.1 创建账户

在我们的管理面板中创建您的账户，并完成基本设置

### STEP.2 对您的爱发电进行设置

登录您的爱发电账号，转到开发者页面  
或直达连接 https://afdian.com/dashboard/dev  

将您的Webhook URL（用来被动接收订单通知）一栏填写  
```json
https://api-dev.b4qaq.cn/callback/aifadian
```
### STEP.3 创建您的应用
来到管理面板中的 APP功能 - 创建APP，创建您的APP  

### STEP.4 ALL Done！
恭喜你已经完成了基础的接入设置，具体的API验证/调用方法请参照 授权API 一章

下文的所有API都基于以上地址构建
## 用户权限
MingChengAPI拥有三种权限：

Veriy - 授权用户  
Admin - 开发者用户  
Root - 网站管理员  

### Verify
MingChengAPI中的基本用户，使用应用名 + Key使用API服务
### Admin
MingChengAPI中的开发者用户，使用用户名 + 密码登录
可以查看/创建/修改自己的APP，并给自己的APP发送公告
::: warning
Admin权限用户无法使用基础API（获取数据API）
:::
### Root
MingChengAPI中的网站管理员，使用用户名 + 密码登录
拥有网站最高权限，可以查看/修改所有数据
::: warning
Root权限用户无法使用基础API（获取数据API）
:::

## 收费标准
- 计费单位：请求数（即每访问一次对应的API）
- 最小计费单位： 不满1个单位的按照一个单位收费
::: warning
本服务为先付费机制，请确保您的剩余请求数充足，请求数不足将直接限制请求
:::

| 服务类别 | 收费标准 |
|--------|------|
| 验证服务 | 免费 |
| 公告服务 | 免费 |
| 定位服务 | 0.1CNY/100次 |
| 天气服务 | 0.1CNY/100次 |


## API错误对照表
::: tip
即便API发生错误时会返回错误原因，但在这里贴张表方便查看
:::
| 状态码 | 说明 |
|--------|------|
| 200 | 操作成功 |
| 201 | Key无效 |
| 202 | 未购买 |
| 400 | 参数错误或操作失败 |
| 401 | 鉴权失败 |
| 403 | 应用未启用相应服务 |
| 404 | 用户未注册 |
| 429 | 访问频率过高 |
| 500 | 服务器内部错误 |

## 请求注意事项

1. 所有API调用都需要确保参数完整，否则会返回400错误。
2. 管理员操作需要提供有效对应APP的管理员Key，否则会返回401错误。
3. 设备相关操作需要提供有效的设备Key，否则会返回401错误。
4. 天气相关API需要对应APP的管理员配置priKey、projectID、docID和weatherAPIURL等信息。
5. 应用必须启用相应服务才能调用相关API，否则会返回403错误。
